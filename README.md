# 🥗 Diet Recommendation System

This project predicts the most suitable diet plan based on user health and lifestyle information using Machine Learning and Streamlit.

---

## 🚀 Live Demo

🔗 https://dietrecommendation-prediction.streamlit.app/

---

## 📌 Project Overview

The Diet Recommendation System helps users receive personalized diet recommendations based on factors like age, gender, BMI, activity level, cholesterol level, sugar level, and fitness goals.

The project uses Machine Learning classification techniques to predict the appropriate diet category.

### 🧠 Input Features

The model takes user inputs such as:

- Age
- Gender
- Height
- Weight
- BMI
- Activity Level
- Sugar Level
- Cholesterol
- Fitness Goal
 ---
 ### 🎯 Output Categories

The model predicts diet types like:

1. Balanced Diet
2. Diabetic Diet
3. High Protein Diet
4. Low Carb Diet
5. Heart Healthy Diet
